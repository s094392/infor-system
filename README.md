# INFOR-System

### Basic useful feature list:

 * IPython notebooks management
 * Workstation management
 * Admin 


### Develop with

 * Node.js
 * Express
 * Docker
 * Socket.io

### How to run
Make sure you have node.js, npm, mongodb installed and mongod is running.
```bash
git clone https://github.com/s094392/infor-system.git
cd infor-system
npm install
npm start
```
